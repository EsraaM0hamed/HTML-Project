% Unit

Defines a measurement as a percentage relative to another value, typically an enclosing element.

em unit is equivalent to the size of a given font

A relative measurement for the height of a font in em spaces, if you assign a font to 12pt, each "em" unit would be 12pt; thus, 2em would be 24pt.

This value defines a measurement relative to a font's x-height.

Defines a measurement in points. A point is defined as 1/72nd of an inch.

px Unit

Defines a measurement in screen pixels.
